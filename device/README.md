# device.js
设备检测，并主动添加到html的className中，方便css控制。

![device.js](https://cloud.githubusercontent.com/assets/2696107/9286775/194403b0-432e-11e5-85ab-1c7801c25119.png)

## 特性

* IsTouch
* IsAndroid
* IsIPad
* IsIPhone
* IsIOS
* IsWinPhone
* IsWebapp
* IsXiaoMi
* IsUC
* IsWeixin
* IsBaiduBox
* IsBaiduBrowser
* IsChrome
* IsPC
* IsHTC
* IsBaiduWallet
* IsDebug
